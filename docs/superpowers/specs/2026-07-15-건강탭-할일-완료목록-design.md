# 배추 건강 탭 할 일 — 완료 목록 표시

날짜: 2026-07-15
배경: 1회성(once) 할 일(예: 가드닐)을 체크하면 "예정" 목록에서 즉시 사라지고, 완료 항목을 다시 볼 수 있는 화면이 없었다. 데이터(done·completedAt)는 DB에 남지만 UI가 없어 실수 체크도 복구 불가.

## 결정
우주 체크리스트(`UjuChecklistList`)와 동일한 패턴으로 **완료 섹션**을 추가한다. (대안이었던 예방접종식 주기 리마인드는 데이터 모델 변경이 필요해 제외 — 재헌님이 완료 목록 방식 선택)

## 범위
수정 파일 1개: `components/baechoo/HealthTodos.tsx`. 데이터 모델·DB·repo 변경 없음.

1. **완료 섹션**: "예정" 카드 아래 `완료 N` 카드. `kind === "once" && done` 항목을 completedAt 최신순 표시 — 체크 켜진 동그라미 + 취소선 제목(stone) + leaf톤 "완료" Pill.
2. **체크 해제**: 완료 항목 체크를 다시 누르면 `done: false, completedAt: null`로 "예정" 복귀(실수 복구).
3. **기존 동작 유지**: 매일 항목 토글, 예정 정렬, 편집 폼(제목 탭) 그대로. `hasAny`에 완료 목록 포함(완료만 있어도 빈 안내문 대신 목록 표시).

기존 완료 데이터(가드닐)는 이 수정만으로 즉시 완료 목록에 나타난다.
