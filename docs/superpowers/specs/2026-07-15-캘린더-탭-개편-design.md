## 2. 가족 캘린더 (서브탭 1, 신규)
### 데이터 모델 — `FamilyEvent` / 테이블 `family_events` (0020)
- `id·title·startDate(YYYY-MM-DD)·endDate(null=당일)·time("HH:MM", null=종일)·assignee(chuchu|baejji|together, TodoAssignee 재사용)·memo·recurrence(none|weekly|monthly)·repeatInterval(1=매주/매월, 2=격주/격월…)·repeatUntil(null=무기한)·exceptions(string[] — 제외한 회차 시작일)·createdAt·deleted_at`
- DB 컬럼: `repeat_interval`(⚠️`interval`은 PG 타입 키워드라 회피)·`repeat_until`·`exceptions jsonb default '[]'`. RLS family_all, start_date idx.
- 반복 모델: **시리즈 단일 레코드 + 예외 날짜 목록**. 편집=시리즈 전체 수정, 특정 회차 제거="이 회차만 삭제"(exceptions에 회차 시작일 push). 구글식 "이후 모두 변경"은 없음(의도된 단순화). 연박 반복이면 회차 길이=endDate−startDate 유지.
### 전개 로직 — `lib/calendar.ts` (순수함수 + vitest)
- `expandEventsInRange(events, fromISO, toISO)` → `EventOccurrence[]{event, date(회차 시작일), dates(포함 날짜들)}`. weekly=7×interval일 스텝, monthly=`addMonths`(월말 클램프, format.ts 재사용). exceptions·repeatUntil 반영.
- `occurrencesByDate(...)` → `Map<dateISO, occurrence[]>` (마커·패널용)
### 화면 — `components/calendar/FamilyCalendar.tsx` + `EventForm.tsx`
- 월 그리드(BaechooCalendar 패턴 확장: 담당자별 마커 색 — 추추 sky/배찌 coral/함께 leaf 계열) + 날짜 클릭 패널 + [+ 일정] 버튼.
- 날짜 패널: ① 그 날 일정(시간순, 종일 먼저 · 편집 탭) ② **이번 주 투두** 섹션(그 날짜 주차의 미완료 WeekTodo, 체크=기존 TodoActionSheet 재사용, 항목 탭=52주 투두 폼)
- dueDate 있는 미완료 투두는 그 날짜에 투두 마커(gold 점).
- EventForm(Sheet): 제목·시작일·종료일(선택)·종일/시간·담당 세그먼트·반복(없음/매주/매월+간격·종료일)·메모·2단계 삭제. 반복 일정 편집 시트에 "이 회차만 삭제" 버튼(패널에서 연 경우, 회차 날짜 전달).
- 조건부 마운트(`{open && <EventForm/>}`) — PlanItemForm 잔존값 함정 회피.
